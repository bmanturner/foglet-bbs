<objective>
Retheme the current widget primitives that are used by the application or present in the widget catalog but are not explicitly overhauled by later v1.3 phases.

Purpose: Phase 17 is the final shared theme-contract phase before visible facelift work begins. Widgets that will not be replaced by Chrome V2, RichRow, EditorFrame, Display.Badge, KvGrid, table presets, inspectors, or screen conversions must be brought onto the semantic theme slot contract here.
Output: focused widget updates plus tests proving Button, Menu, SmartList, Display.Progress, Tabs, RadioGroup, Checkbox, SelectionList, Spinner, and generic Modal do not drift from the Phase 17 theme contract.
</objective>
