## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Operator shell -> Mix task | CLI args select actor, target, and target status. |
| Mix task -> Accounts | Break-glass path crosses into authoritative status mutation. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-10-11 | Elevation of Privilege | `mix foglet.user.status` | mitigate | Require `--actor SYSOP_HANDLE` and call `Accounts.transition_user_status/3`; no task-local bypass. |
| T-10-12 | Tampering | status parsing | mitigate | Allowlist only `"active"`, `"rejected"`, `"suspended"` before `String.to_existing_atom/1`. |
| T-10-13 | Repudiation | operator output | mitigate | Print exact status change and delivery outcome for every success. |
| T-10-14 | Tampering | deleted users | mitigate | Surface Accounts `{:error, :deleted}` as `"Deleted users cannot be changed."` and exit non-zero. |
| T-10-15 | Denial of Service | bad CLI args | mitigate | Invalid/missing arguments exit before any Accounts mutation. |
</threat_model>
