| ThreadList route params -> PostReader/NewThread | Selected board/thread identity crosses screen boundaries. |
| Task result -> ThreadList reducer | Async thread rows update screen-local state. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-36-04 | Elevation of Privilege | Thread load task effect | mitigate | Keep thread visibility and unread annotation in `Foglet.Threads`; ThreadList only requests the context API. |
| T-36-05 | Tampering | Route params for selected board/thread | mitigate | Reducer tests assert params are derived from loaded rows/local board state, not freeform user input. |
| T-36-06 | Denial of Service | Thread sorting and render | mitigate | Sorting remains in-memory over already loaded rows; DB work stays inside task effects. |
