<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| keyboard input -> Account draft state | Untrusted terminal input changes local draft values. |
| Account draft state -> render theme override | User-selected theme candidate affects rendering before persistence. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-08 | Tampering | Account forms | mitigate | Drafts are local until explicit save; cancel/revert reseeds from `current_user`. |
| T-05-09 | Tampering | Theme preview | mitigate | Candidate theme affects Account rendering only; do not mutate `session_context` or Session state before save. |
| T-05-10 | Elevation of Privilege | Account save command payload | mitigate | Forms emit only profile/preference attrs: location, tagline, real_name, timezone, preferences.time_format, and theme. |
| T-05-11 | Information Disclosure | Profile form | accept | `real_name` is private but only rendered to the current user's Account screen. |
</threat_model>
