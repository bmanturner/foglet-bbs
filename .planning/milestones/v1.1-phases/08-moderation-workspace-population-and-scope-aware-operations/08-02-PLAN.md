| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-01 | Spoofing | Workspace snapshot actor | mitigate | Use `Foglet.Authorization.scopes_for/2`; empty scopes cannot receive populated data. |
| T-08-02 | Tampering | Moderation tabs | mitigate | Tabs in this plan are read-only; no caller-supplied mutation ids are accepted. |
| T-08-03 | Repudiation | LOG tab | mitigate | Render durable `mod_actions` rows created by Plan 08-01, newest first. |
| T-08-04 | Information Disclosure | `workspace_snapshot/1` and render state | mitigate | Scope list controls all populated data; regular users/guests render only not-available copy. |
| T-08-05 | Denial of Service/Layout | log/user/board text | mitigate | Limit snapshot row counts to 20 and truncate reason/body/handle strings in rendering. |
</threat_model>
