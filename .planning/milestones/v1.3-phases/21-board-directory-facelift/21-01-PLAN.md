<objective>
Add a `:last_post_at` field to the `Foglet.Boards.directory_board` map returned by `board_directory_for/1`. The value is the maximum `Foglet.Threads.Thread.last_post_at` across non-deleted threads on each board, or `nil` when the board has no non-deleted threads. The value is actor-independent (boards are public-readable per SPEC line 140).

Purpose: Phase 21's BoardTree widget renders an age column on every board row. Without this field, the row cannot show "12m" / "2h" / "—". Per CONTEXT D-09, the data must be derived in a single LEFT JOIN aggregate query (no N+1).

Output: Extended `directory_board` typespec, new private `last_post_ats/0` helper rooted on `Board` (not `Subscription` — see Pitfall 4), and ExUnit cases under the existing `describe "board_directory_for/1 (SUBS-01)"` block proving correctness for subscribed, unsubscribed, deleted-thread, and no-thread cases.

Implements requirement: BOARDS-03 (`Foglet.Boards.board_directory_for/1` exposes `:last_post_at`).

@spec unread_counts(String.t()) :: %{String.t() => non_neg_integer()}
def unread_counts(user_id) do
  Repo.all(
    from s in Subscription,
      where: s.user_id == ^user_id,
      left_join: rp in ReadPointer,
      on: rp.user_id == s.user_id and rp.board_id == s.board_id,
      left_join: p in Foglet.Posts.Post,
      on:
        p.board_id == s.board_id and
          p.message_number > coalesce(rp.last_read_message_number, 0) and
          is_nil(p.deleted_at),
      group_by: s.board_id,
      select: {s.board_id, count(p.id)}
  )
  |> Map.new()
end
