<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Effect interpreter -> App state | Generic effects can change route, modal, terminal, and session runtime state. |
| Task command -> screen reducer | Async results re-enter the screen update loop after off-process execution. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-34-04 | Tampering | App.apply_effect/2 | mitigate | Tests must prove navigation initializes only the target screen-state key and preserves unrelated legacy fields. |
| T-34-05 | Denial of Service | Task effects | mitigate | Task effects must use Foglet.TUI.Command.task/2 so exceptions become error messages instead of crashing the lifecycle. |
| T-34-06 | Repudiation | Task result routing | accept | Phase 34 routes reducer messages; durable audit/logging remains with domain contexts and later screen migrations. |
</threat_model>
