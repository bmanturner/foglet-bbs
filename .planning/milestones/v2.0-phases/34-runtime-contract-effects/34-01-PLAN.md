<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Screen reducer -> App runtime | Screen-produced effects cross into App-owned runtime state and command execution. |
| App runtime -> task closure | Task effects contain functions that run off-process through Raxol commands. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-34-01 | Information Disclosure | Foglet.TUI.Context | mitigate | Context tests must prove App screen-storage fields are not public fields. |
| T-34-02 | Tampering | Foglet.TUI.Effect.task/3 | mitigate | Constructor guards require atom operation names and zero-arity functions; the function is not executed by construction. |
| T-34-03 | Elevation of Privilege | Screen contract | accept | Phase 34 adds pure TUI runtime contracts only; domain authorization remains in Foglet contexts and is not weakened. |
</threat_model>
