`state.posts`, `state.read_position`, or `screen_state[:post_reader]` fields
itself.

| Route/local state -> PubSub subscription | Thread subscriptions derive from visible route or local screen identity. |

| T-37-05 | Information Disclosure | Thread PubSub topics | mitigate | Subscribe only to the routed/local thread id for the visible PostReader route. |
| T-37-06 | Denial of Service | Thread activity refresh | mitigate | Active-thread refresh emits the same single PostReader load task path as route entry. |
