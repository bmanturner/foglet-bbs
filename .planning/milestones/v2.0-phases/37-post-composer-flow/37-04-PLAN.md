## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| NewThread reducer -> Boards context | Board picker load task reads subscribed boards. |
| NewThread reducer -> Threads context | Create-thread task writes a thread and opening post. |
| NewThread -> ThreadList route params | New thread selection intent crosses screens through route params. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-37-10 | Elevation of Privilege | Thread creation | mitigate | Keep authorization and message-number allocation inside `Foglet.Threads.create_thread/3` and `Foglet.Boards.Server`. |
| T-37-11 | Tampering | Board selection | mitigate | Submit uses the local selected board id from loaded board data; tests assert exact create_thread args. |
| T-37-12 | Denial of Service | Board load and create-thread paths | mitigate | Both domain operations run through `Effect.task/3`, not synchronously in key handling. |
</threat_model>
